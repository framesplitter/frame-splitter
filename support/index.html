Support

Last updated: June 2, 2026

This page provides support information for Frame Splitter, a Canva app that detects and splits frames from storyboard images.

What Frame Splitter does

Frame Splitter helps you upload a storyboard image, detect frame areas, manually adjust detection boxes if needed, and place the extracted frames into your Canva design in order.

Supported file types

Frame Splitter supports the following image formats:

  • PNG
  • JPG / JPEG
  • WEBP

Basic usage

  1. Open Frame Splitter inside a Canva design.
  2. Upload a storyboard image.
  3. Adjust frame detection settings if needed.
  4. Review the detection preview.
  5. Manually adjust or add frame areas if necessary.
  6. Select the output aspect ratio and size.
  7. Set the placement size.
  8. Place the extracted frames into your Canva design.

Troubleshooting

Frames are not detected correctly

Try adjusting the margin or frame sensitivity settings. You can also manually add, move, or resize detection boxes in the detection preview.

The extracted frames are not in the expected order

Frame Splitter generally sorts frames from top-left to bottom-right. If needed, use the ordering controls to move each cut up or down.

The output size is not what I expected

Check the selected aspect ratio and output size settings before placing the frames. Frame Splitter supports 16:9, 9:16, and 1:1 output presets.

The app is not responding

Try closing and reopening the app inside Canva. If the issue continues, please contact us with a brief description of the issue and the image format you were using.
